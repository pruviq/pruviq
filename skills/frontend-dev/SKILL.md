# Frontend Dev Skill (overview)

This skill contains procedures, scripts, and references for frontend tasks. Load the bundled resources only when the task touches code under `src/`, `styles/`, or `layouts/`.

Resources (examples):
- scripts/build-preview.sh — build + produce screenshots
- references/styleguide.md — design tokens and Tailwind guidance

When to use:
- Fixing visual regressions reported by QA
- Implementing new UI features or responsive adjustments
- Improving Lighthouse/SEO related frontend changes

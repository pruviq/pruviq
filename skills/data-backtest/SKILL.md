# Data & Backtest Skill

Include scripts to run reproducible backtests and validation checks under `scripts/backtest/` as bundled resources.

- Data inconsistencies in public/data/ or API responses
- Re-running/backtesting strategies for reproducibility (create reproducible package)
- Generating demo datasets for documentation or reproducible reports

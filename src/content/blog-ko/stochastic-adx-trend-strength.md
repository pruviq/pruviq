## 두 가지 질문, 두 가지 지표

모든 트레이드는 두 가지 질문에 답해야 합니다:
1. **모멘텀은 어디인가?** → 스토캐스틱 오실레이터
2. **추세는 얼마나 강한가?** → ADX

이 지표들은 상호 보완적입니다. 스토캐스틱은 *방향*을, ADX는 *강도*를 알려줍니다.

## 스토캐스틱 오실레이터

현재 가격이 최근 고가-저가 범위에서 어디에 있는지 측정합니다:

```
%K > 80  → 최근 고점 근처 (과매수)
%K < 20  → 최근 저점 근처 (과매도)
```

### 주요 신호
- **과매도 반등**: %K가 20 아래에서 %D 위로 교차 → 롱 가능성
- **과매수 거부**: %K가 80 위에서 %D 아래로 교차 → 숏 가능성

## ADX 설명

ADX는 방향과 관계없이 **추세의 강도**를 측정합니다:

```
ADX > 25  → 강한 추세 (추세 시장)
ADX < 20  → 약한/무추세 (횡보 시장)
ADX > 40  → 매우 강한 추세 (역행하지 마세요)

+DI > -DI → 상승 추세
-DI > +DI → 하락 추세
```

### ADX가 중요한 이유

트레이딩의 가장 큰 실수는 잘못된 시장에 잘못된 전략을 적용하는 것:
- **추세 시장 + 평균회귀 전략** = 손실
- **횡보 시장 + 추세추종 전략** = 휩소

ADX가 이 불일치를 피하게 해줍니다.

## PRUVIQ에서 활용

| 스토캐스틱 필드 | 설명 |
|---------------|------|
| `stoch_k` | %K 값 (0-100) |
| `stoch_d` | %D 신호 라인 |
| `oversold` | %K < 20일 때 True |
| `overbought` | %K > 80일 때 True |

| ADX 필드 | 설명 |
|----------|------|
| `adx` | ADX 값 (0-100) |
| `plus_di` | +DI (상승 방향) |
| `minus_di` | -DI (하락 방향) |
| `trend_strength` | ADX > 25면 "strong" |

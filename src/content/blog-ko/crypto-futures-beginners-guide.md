무기한 선물은 크립토에서 가장 많이 거래되는 상품으로, 일일 거래량이 $1,000억을 넘습니다. 하지만 대부분의 초보자가 돈을 잃는 이유는 레버리지, 마진, 청산이 실제로 어떻게 작동하는지 이해하지 못하기 때문입니다.

## 무기한 선물이란?

무기한 선물 계약은 실제 자산을 소유하지 않고 암호화폐의 가격이 오를지(롱) 내릴지(숏) 베팅할 수 있게 해줍니다. 전통적인 선물과 달리 만기일이 없어 "무기한"으로 거래됩니다.

**현물 거래와의 핵심 차이**: 현물에서는 BTC 1개를 $70,000에 사서 나중에 팝니다. 선물에서는 가격 움직임에 따라 수익이나 손실이 발생하는 포지션을 열며, 레버리지로 증폭됩니다.

**왜 존재하는가**: 숏(가격 하락 시 수익), 레버리지(빌린 돈으로 거래), 기초자산 보유 없이 거래가 가능합니다.

## 레버리지 작동 원리

레버리지는 노출을 배가합니다. 5배 레버리지와 $100 마진으로:

- $500 포지션을 통제
- 가격 +10% 이동 시 $50 수익 ($100의 50%)
- 가격 -10% 이동 시 $50 손실 ($100의 50%)
- -20%에서 $100 전액 손실 (마진의 100%) → **청산**

| 레버리지 | $100 마진 통제액 | 청산까지 거리 |
|----------|----------------|-------------|
| 1배 | $100 | -100% (크립토에서 사실상 불가) |
| 5배 | $500 | -20% |
| 10배 | $1,000 | -10% |
| 20배 | $2,000 | -5% |
| 50배 | $5,000 | -2% |
| 100배 | $10,000 | -1% |

**현실**: 100배 레버리지에서는 1%의 불리한 가격 움직임이 마진 전액을 날립니다. 크립토에서 1% 움직임은 몇 분 만에 발생합니다.

## 펀딩비란?

무기한 선물은 만기가 없으므로 가격을 현물가에 가깝게 유지하는 메커니즘이 필요합니다. 그것이 **펀딩비**입니다.

- **양의 펀딩비**: 롱이 숏에 지불 (선물가가 현물가 위 — 강세 심리)
- **음의 펀딩비**: 숏이 롱에 지불 (선물가가 현물가 아래 — 약세 심리)
- **지불 주기**: 대부분 거래소에서 8시간마다 (바이낸스: 00:00, 08:00, 16:00 UTC)
- **일반적인 비율**: 8시간당 0.01% (~일 0.03%, ~연 10.95%)

**왜 중요한가**: 포지션을 며칠 또는 몇 주 보유하면 펀딩비가 쌓입니다. 일 0.03% 펀딩비는 월 ~1%에 해당합니다. 레버리지 포지션에서는 상당할 수 있습니다.

## 청산 작동 원리

미실현 손실이 마진에 근접하면, 거래소가 포지션을 강제 종료합니다. 이것이 청산입니다.

**격리 마진**: 각 포지션에 별도 마진. 포지션 A가 청산되어도 포지션 B에 영향 없음. 리스크 관리에 더 안전.

**교차 마진**: 모든 포지션이 하나의 마진 풀 공유. 자본 효율적이지만 더 위험 — 하나의 나쁜 포지션이 모든 것을 청산할 수 있음.

**청산가 예시** (격리, 5배 레버리지, SHORT):
- 진입가: $70,000
- 마진: $100 ($500 포지션)
- 약 $84,000에서 청산 (불리한 방향 +20% 이동)

## 가장 흔한 실수

### 실수 1: 과도한 레버리지 사용

초보자는 100배 레버리지를 보고 "100배 수익을 낼 수 있다"고 생각합니다. 실제로는: 1% 움직임에 청산됩니다.

**우리의 접근**: 5배 레버리지를 사용합니다. 청산 전 20%의 여유를 확보하며, 알트코인 시간봉 포지션에 합리적입니다.

### 실수 2: 손절 미설정

"다시 돌아올 거야"는 트레이딩에서 가장 비싼 문장입니다. 손절 없이는 포지션이 하나의 캔들에서 -5%에서 -100%로 갈 수 있습니다.

**우리의 접근**: 모든 포지션에 진입 전 손절을 설정합니다. 현재 BB Squeeze 전략에서 10%. 예외 없음, 수동 개입 없음.

### 실수 3: 과대한 포지션 사이즈

계좌의 50%를 한 거래에 넣으면 한 번의 나쁜 거래가 자본 절반을 날립니다.

**예시**: $10,000 자본에 포지션당 $200 = 거래당 2%. 10연패도 20%만 비용.

### 실수 4: 수수료 무시

선물 거래 수수료는 레버리지와 함께 빠르게 누적됩니다:

- 바이낸스 선물: 메이커 0.02% / 테이커 0.04% (BNB 할인 포함)
- 5배 레버리지: 실효 수수료 = 마진당 왕복 0.1-0.2%
- 월 100건 거래 시 0.15% = 자본의 15%가 수수료로 소진

**수수료 절감 방법**:
- 시장가 주문(테이커) 대신 지정가 주문(메이커) 사용
- BNB로 수수료 결제 (바이낸스 25% 할인)
- [추천 코드로 추가 할인](/ko/fees)

## 안전하게 시작하기

1. **페이퍼 트레이딩부터**: 대부분의 거래소가 테스트넷 또는 페이퍼 트레이딩 제공. 여기서 먼저 연습.
2. **격리 마진 사용**: 포지션당 리스크 제한.
3. **낮은 레버리지 유지**: 초보자는 최대 3-5배. 프로 알고 트레이더도 10배 이상은 드물게 사용.
4. **즉시 손절 설정**: 포지션 진입 전, 최대 허용 손실을 파악.
5. **소액 시작**: 첫 100건 거래는 수업료. 최소 포지션 사이즈 사용.
6. **모든 것을 기록**: 모든 거래, 이유, 결과를 기록. 측정하지 않으면 개선할 수 없습니다.

## 거래소 선택

거래소가 중요합니다. 수수료, 유동성, 레버리지 옵션, 안정성이 모두 다릅니다.

| 요소 | 중요한 이유 |
|------|-----------|
| 수수료 | 월 100건+ 거래 시 0.01% 차이 = 실질적 금액 |
| 유동성 | 낮은 유동성 = 넓은 스프레드 = 나쁜 체결 |
| API 안정성 | 알고 트레이딩에서 API 중단 = 놓친 거래 |
| 코인 수 | 더 많은 코인 = 더 많은 전략 기회 |

우리는 바이낸스 선물을 사용합니다 (최고 유동성, 500개 이상 무기한 페어). 대안은 [전체 거래소 비교](/ko/fees)를 참고하세요.

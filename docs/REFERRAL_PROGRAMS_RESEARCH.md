## 1. Binance (바이낸스)

### 프로그램 개요
- **일반 레퍼럴**: 누구나 가능, 즉시 발급 (KYC 필요)
- **아피리에이트 파트너**: 심사 필요, 고급 기능 + 높은 커미션

### 등록 페이지
- **일반 레퍼럴**: https://www.binance.com/en/activity/referral
- **아피리에이트**: https://www.binance.com/en/activity/affiliate

### 커미션 구조

#### 일반 레퍼럴 프로그램
- **현물**: 20-40% (BNB 보유량 기준)
  - < 500 BNB: 20% 커미션
  - ≥ 500 BNB: 40% 커미션
- **선물**: 10-20%
- **킥백 옵션**: 0%, 5%, 10%, 15%, 20% 중 선택 가능

#### 아피리에이트 프로그램
- **최대 커미션**: 50%
- **추가 혜택**: 월 최대 $72,000 보너스 (거래량 기준)
- **제공 도구**: 트래킹 대시보드, 마케팅 자료, 전담 지원팀

### 사용자 할인
- **표준 코드**: 10% 거래 수수료 할인
- **프리미엄 코드**: 15-20% 할인
- **2025년 7월 업데이트**: 향상된 커미션 구조 적용

### 레퍼럴 링크 형식
```
https://accounts.binance.com/register?ref=[REFERRAL_CODE]

예시: https://accounts.binance.com/register?ref=BESTFUTURES
```

### 승인 절차
- **일반 레퍼럴**: KYC 완료 시 즉시 발급
- **아피리에이트**:
  - 신청서 제출 (콘텐츠, 오디언스, 홍보 계획)
  - 7-14일 심사 기간 (성수기 더 길 수 있음)
  - 콘텐츠 품질 및 프로필 진정성 평가

### 필수 요건
- **KYC 인증**: 필수 (2026년 정책 - 레퍼러/레퍼리 모두)
- **아피리에이트**:
  - 증명 가능한 오디언스
  - 트래픽/팔로워 수 (구체적 수치 미공개)
  - 관련성 있는 콘텐츠

## 가격은 거짓말합니다. 거래량은 아닙니다.

가격은 조작될 수 있습니다. 단일 대형 주문으로 일시적으로 밀어올리거나 내릴 수 있습니다. 하지만 지속적인 거래량은 위조하기 어렵습니다 — 많은 트레이더의 실제 참여를 나타냅니다.

```
높은 거래량 + 가격 움직임  = 진짜 움직임 (추세 지속 가능성)
낮은 거래량 + 가격 움직임  = 의심스러움 (페이크아웃 가능성)
거래량 급등 + 가격 변화 없음 = 축적/분배 진행 중
```

## 거래량 지표 설명

### 거래량 비율 (Volume Ratio)

가장 실용적인 거래량 지표. 현재 거래량을 평균과 비교합니다:

```
Volume Ratio = 현재 거래량 / SMA(거래량, 20)

비율 > 2.0  → 평균의 2배 (유의미)
비율 > 3.0  → 평균의 3배 (매우 유의미)
비율 = 1.0  → 평균 (특별할 것 없음)
```

PRUVIQ의 검증된 전략은 **거래량 비율 >= 2.0**을 필수 필터로 사용합니다. 백테스트에서 이 필터 하나가 거짓 신호를 ~40% 줄였습니다.

### Volume Z-Score

현재 거래량이 얼마나 비정상적인지를 통계적으로 측정:

```
Z > 2.0  → 정규분포 2 표준편차 이상 (희귀 이벤트)
Z > 3.0  → 극도로 비정상적 (확률 < 0.3%)
```

청산 캐스케이드, 뉴스 이벤트, 고래 축적 등 "무언가 일어나고 있다"는 순간을 포착합니다.

## 크립토의 거래량 패턴

### 패턴 1: 거래량이 가격에 선행
대형 거래량은 가장 큰 가격 움직임 *이전*에 나타나는 경향이 있습니다.

### 패턴 2: 거래량이 돌파를 확인
3배 거래량의 돌파는 1배 거래량보다 신뢰할 수 있습니다:
- vol_ratio >= 2.0 돌파: **59% 승률**
- vol_ratio < 2.0 돌파: **48% 승률**

11%p의 차이가 엣지입니다.

## MACD란?

MACD(이동평균수렴확산)는 두 EMA 사이의 관계를 측정합니다:

```
MACD 라인  = EMA(12) - EMA(26)
신호 라인  = MACD 라인의 EMA(9)
히스토그램 = MACD 라인 - 신호 라인
```

MACD 라인이 신호 라인 위에 있으면 모멘텀은 상승, 아래면 하락. 히스토그램은 둘 사이의 간격을 보여줍니다 — 커지는 히스토그램은 모멘텀 가속을 의미.

## 세 가지 MACD 신호

### 1. 크로스오버
MACD가 신호 라인 위로 → 상승 / MACD가 신호 라인 아래로 → 하락

### 2. 제로 라인 교차
MACD가 0 위로 → 추세 상승 전환 / MACD가 0 아래로 → 추세 하락 전환

### 3. 다이버전스
가격은 신고가, MACD는 더 낮은 고점 → 하락 다이버전스 (모멘텀 약화)

## 크립토에서 MACD의 현실

535개 코인에서 MACD 기반 전략을 테스트한 결과:

**작동하는 것:**
- **모멘텀 필터**로서의 MACD — 다른 신호 확인용
- 히스토그램 방향 — 모멘텀 가속/감속 표시
- 제로 라인 위치 — 빠른 추세 방향 확인

**작동하지 않는 것:**
- 독립적 진입 신호로서의 MACD 크로스오버 — 너무 느리고 거짓 신호 많음
- 다이버전스만으로 트레이딩 — 타이밍 불안정

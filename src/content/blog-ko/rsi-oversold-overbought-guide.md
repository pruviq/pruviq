## RSI가 실제로 측정하는 것

RSI(Relative Strength Index)는 가격 변동의 속도와 크기를 측정하는 모멘텀 오실레이터입니다. 0에서 100 사이의 값을 출력합니다.

```
RSI > 70  → "과매수" — 가격이 너무 빠르게 상승했을 수 있음
RSI < 30  → "과매도" — 가격이 너무 빠르게 하락했을 수 있음
RSI = 50  → 중립 모멘텀
```

대부분의 트레이더는 RSI를 단순한 매수/매도 신호로 배웁니다: 과매도일 때 매수, 과매수일 때 매도. 교과서적 해석입니다. 현실은 더 복잡합니다.

## 계산 공식

```
RS = 평균 상승폭 / 평균 하락폭 (기본 14기간)
RSI = 100 - (100 / (1 + RS))
```

1시간 차트에서 14기간 RSI는 최근 14개 캔들을 분석합니다. 대부분이 상승이면 RSI가 높고, 하락이면 낮습니다.

## 크립토에서 RSI가 작동할 때

RSI는 **횡보장**에서 가장 잘 작동합니다 — 가격이 지지선과 저항선 사이에서 진동할 때.

- **과매도 반등**: 횡보장에서 RSI < 30은 단기 회복의 전조
- **과매수 거부**: RSI > 70은 풀백 전 소진 신호
- **다이버전스**: 가격은 신고가, RSI는 더 낮은 고점 → 모멘텀 약화

## RSI가 실패할 때

**강한 추세**에서 RSI는 장기간 과매수/과매도 상태를 유지합니다. 상승장에서 RSI > 70이 수주간 유지되는 것은 정상이며, 매도 신호가 아닙니다.

따라서 RSI만으로는 트레이딩 신호가 되지 않습니다. 맥락이 필요합니다:
- **추세 방향** — 시장이 추세인가 횡보인가?
- **거래량 확인** — 실제 자금이 움직임 뒤에 있는가?
- **다중 타임프레임** — 일봉 RSI와 시간봉 RSI는 다른 이야기

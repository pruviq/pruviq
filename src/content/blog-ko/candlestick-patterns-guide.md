## 캔들스틱 기초

각 캔들스틱은 하나의 시간 구간에 대한 4개의 가격을 보여줍니다:

```
양봉 (녹색):             음봉 (적색):
    ─── 고가                 ─── 고가
    │                        ┌──┐
    ┌──┐ ← 종가              │  │ ← 시가
    │  │                     │  │
    │  │ ← 시가              └──┘ ← 종가
    └──┘                        │
    │                        ─── 저가
    ─── 저가
```

몸통은 시가-종가 범위, 꼬리(그림자)는 고가와 저가 극단을 보여줍니다.

## 단일 캔들 패턴

### 도지 (Doji, 비결정)
시가와 종가가 거의 같음. 매수세도 매도세도 이기지 못한 상태. 종종 잠재적 반전을 시사하지만 맥락이 필요합니다.

**크립토 현실**: 짧은 시간대에서 끊임없이 출현. 단독 도지는 예측 가치가 거의 없음. 극단(강한 추세 후, 핵심 레벨)에서만 의미 있음.

### 해머 / 교수형
긴 아래꼬리, 윗부분에 작은 몸통. 해머는 하락추세 후 출현(상승 신호). 교수형은 상승추세 후 출현(하락 신호).

**크립토 현실**: 일봉에서 어느 정도 유용. 긴 꼬리가 낮은 가격에 대한 거부를 보여줌. 거래량과 지지 레벨과 결합 시 더 효과적.

### 유성형 (Shooting Star)
해머의 반대. 긴 위꼬리가 높은 가격에 대한 거부를 보여줌.

## 복합 캔들 패턴

### 장악형 (Engulfing)
두 번째 캔들이 첫 번째를 완전히 "감싸는" 패턴. 상승 장악형 = 양봉이 이전 음봉을 감쌈. 반전 신호.

**크립토 현실**: 상위 시간대의 핵심 지지/저항에서 발생 시 비교적 신뢰할 수 있는 패턴. 여전히 확인 필요.

### 샛별 / 저녁별 (Morning Star / Evening Star)
3개 캔들 반전 패턴:
1. 추세 방향의 큰 캔들
2. 작은 몸통 캔들 (비결정)
3. 반대 방향의 큰 캔들

## 백테스트 결과

535+ 코인, 1시간 봉에서 캔들스틱 패턴 인식을 실행한 결과:

| 패턴 | 승률 (단독 사용) | 판정 |
|------|-----------------|------|
| 도지 | ~50% | 동전 던지기와 다를 바 없음 |
| 해머 | ~52% | 약간의 엣지, 확인 필요 |
| 장악형 | ~53% | 단독 최고, 여전히 한계 |
| 샛별/저녁별 | ~54% | 괜찮지만 발생 빈도 낮음 |
| 적삼병 | ~51% | 크립토에서 비신뢰 |

**불편한 진실**: 크립토에서 단독 사용 시 거래 가능한 엣지를 만드는 캔들스틱 패턴은 없습니다. 승률이 랜덤에 겨우 앞섭니다.

## 패턴이 가치를 더하는 경우

캔들스틱 패턴은 시그널이 아닌 **확인**으로 작동합니다:

1. **핵심 레벨에서**: 지지에서의 해머 = 의미 있음. 중간에서의 해머 = 잡음
2. **거래량과 함께**: 3배 평균 거래량의 장악형 = 강함. 저거래량 = 약함
3. **상위 시간대에서**: 역대 최고가에서 일봉 도지 = 주목할 만함. 5분봉 도지 = 무의미
4. **강한 움직임 후**: 반전 패턴은 반전할 대상이 필요

## PRUVIQ에서 캔들스틱 활용

BB Squeeze 전략은 간단하지만 효과적인 캔들스틱 요소를 사용합니다:

```
음봉 (종가 < 시가) = SHORT 진입 확인
```

복잡한 패턴이 아닌 기본적인 방향 필터. 이 단순함이 정교한 패턴 인식보다 종종 더 잘 작동합니다.

## 핵심 요약

> 캔들스틱 패턴은 매수/매도 역학을 이해하는 시각적 언어이지 예측 도구가 아닙니다. 핵심 레벨에서 거래량과 함께 확인으로 사용하면 가치를 더하지만, 단독으로는 랜덤에 겨우 앞섭니다. 크립토에서는 단순한 것이 보통 더 좋습니다.
